# 校准历史 (Calibration Log)

Phase 3 校准历史记录。每次 biz-impact-analysis 完成后由 risk-classifier Phase 3 自动追加。

> **本文件角色**: 完整维度对比日志（详细表格），供人工 review 和偏差模式分析。
> **对应文件**: `state/calibration-history.md` 是简洁结构化索引，供 Phase 1 自动快速检索。
>
> 积累足够记录后，可分析系统性偏差模式（如"某域总是被低估"），用于调整 `change-risk-classification.md` 中的因子权重。
